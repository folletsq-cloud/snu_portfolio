# 문현경 포트폴리오 사이트

음악교육자 문현경의 프로필, 핵심 역량, 자격, 경력, 학력 정보를 담은 정적 포트폴리오 사이트입니다.

## 구성

- `index.html`: 사이트 마크업
- `styles.css`: 전체 디자인과 반응형 스타일
- `script.js`: 간단한 인터랙션 스크립트
- `docs/snuhk.md`: 기존 이력서 원문 정리본
- `vercel.json`: Vercel 정적 배포 설정

## 로컬 확인

별도 빌드 과정 없이 `index.html`을 브라우저에서 열어 확인할 수 있습니다.

## Vercel 배포

Vercel에서 이 폴더를 프로젝트로 연결하면 정적 사이트로 배포할 수 있습니다.

- Framework Preset: `Other`
- Build Command: 비워둠
- Output Directory: 비워둠
